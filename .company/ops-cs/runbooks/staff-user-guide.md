# 📱 イベント管理アプリ 使い方ガイド（スタッフ用）

EX事業部 イベント管理システム「**Ivent Manager**」へようこそ！
このガイドは、現場スタッフのみなさんが毎日アプリを使うための説明書です。スマホ・PCどちらでも使えます。

**最終更新**: 2026-06-01 ／ 困ったときの連絡先：**taoki0183@gmail.com**

---

## 1. はじめに ― ログインする

1. ブラウザ（スマホは **Safari** または **Chrome**）でアプリのURLを開きます。
2. ログイン画面で、次のいずれかでログインします。
   - **Googleでログイン**（おすすめ）
   - **Appleでログイン**
   - **メールアドレスでログイン**（事前にパスワード登録が必要）
3. ログインできれば完了です。

> ⚠️ **「アクセス権限がありません」と表示されたら？**
> あなたのアカウントがまだ登録されていません。画面の「管理者に連絡する」から、ログインに使ったメールアドレスを添えて連絡してください。登録後、再度ログインできます。

---

## 2. ホーム画面に追加する（おすすめ・1分）

毎回ブラウザで開かなくても、**ホーム画面のアイコンから一発で起動**できます。アプリのように全画面で使えて便利です。

### iPhone（Safari）
1. アプリを Safari で開く
2. 下の**共有ボタン**（□から↑が出ているマーク）をタップ
3. メニューを下にスクロールして「**ホーム画面に追加**」をタップ
4. 右上の「追加」をタップ → ホーム画面に「Ivent Manager」アイコンが追加されます

### Android（Chrome）
1. アプリを Chrome で開く
2. 右上の「**︙**（メニュー）」をタップ
3. 「**ホーム画面に追加**」または「**アプリをインストール**」をタップ
4. 「追加／インストール」で完了

---

## 3. 画面の使い方

下のメニュー（PCは上のメニュー）で画面を切り替えます。

| 画面 | できること |
|------|-----------|
| 🏠 **ホーム** | 直近のイベント・開催中・期限が近いものをまとめて確認 |
| 📅 **カレンダー** | 月ごと・日ごとにイベントを一覧。会場やクライアントで**検索**も可 |
| 📋 **準備物リスト** | イベントごとの持ち物・備品をチェック管理（後述） |
| 🗄️ **アーカイブ** | 終了したイベントの記録 |
| 📦 **備品マスター** | よく使う備品のひな型を管理 |
| 🐟 **魚リスト** / 🧩 **レイアウト** | 水族館イベントの魚の数・会場レイアウト管理 |

検索バーに**会場名やクライアント名**を入れると、すばやく目的のイベントが見つかります。

## 4. 準備物リストを使う（全員が編集できます）

イベントを開いて「準備物リスト」を見ると、持ち物・備品を管理できます。

- **➕ 項目を追加**：「新しい項目を追加」をタップ
- **品名・数量・単価・配送料・URL・備考**を入力 → 金額は自動計算
- **✅ 到着 / 準備完了**のチェックで進捗を管理
- 画面下に**合計金額**と**進捗％**が表示されます

> ✨ **自動保存になりました！**
> チェックや入力は**自動で保存**されます（「保存」ボタンを押す必要はありません）。
> 画面上部に「**保存中… → ✓ 自動保存済み**」と表示されるので、それを確認すればOK。
> ※ 以前の「チェックしたのに消えた」問題は解消されています。

### 便利な出力機能
- 🖨️ **印刷**：準備物リストをそのまま印刷
- 📊 **Excel出力**：Excelファイルとしてダウンロード
- 💼 **商談提案**：見積り風のPDFを作成／**LINEで共有**も可能

## 5. 写真をアップロードする（全員が編集できます）

イベント詳細の「写真」タブから、現場の写真を記録できます。

- 1イベントあたり**最大5枚**まで
- 1枚あたり**10MBまで**（写真形式：jpg / png / heic など）
- アップロードした写真はみんなで共有・閲覧できます

> 写真が上がらないときは、**サイズが10MB以下か**・**画像ファイルか**を確認してください。

---

## 6. 「編集できる人」について

安全のため、操作によって権限が分かれています。

| 操作 | 誰ができる？ |
|------|------------|
| イベントの閲覧・検索 | ログインした全員（スマホ・PC両方） |
| **準備物リストの編集** | ログインした全員（スマホ・PCどちらも◎） |
| **写真のアップロード** | ログインした全員 |
| **イベント本体の作成・編集・削除** | 一部の編集担当者のみ・**PCのみ**（スマホ不可） |

> 📌 **「スマホでイベントを編集できない」のは仕様です。**
> 誤操作防止のため、イベント本体（日程・会場など）の編集は**PCの編集担当者のみ**に限定しています。
> 準備物リストと写真は、スマホでも全員編集できます。

## 7. よくある質問（FAQ）

| 困りごと | 対処 |
|---------|------|
| ログインしても弾かれる | アカウント未登録です。「管理者に連絡する」からメールアドレスを伝えて登録を依頼してください |
| スマホでイベントを編集できない | 仕様です。準備物・写真はスマホでも編集できます。イベント本体はPCの担当者へ依頼してください |
| 準備物のチェックは保存される？ | **自動保存**されます。上部の「✓ 自動保存済み」表示を確認してください |
| 写真がアップロードできない | サイズ（10MB以下）と画像形式を確認してください |
| 表示名を変えたい | Googleアカウントのプロフィール名を変更後、アプリに再ログインしてください |
| 画面が真っ白／動かない | 一度アプリを完全に閉じて再起動、改善しなければ管理者へ連絡 |
